# ShopMart

ShopMart is a lightweight, responsive online shopping application built with **Bootstrap 5** and **Vanilla JavaScript**. It demonstrates a full client-side e-commerce flow using `LocalStorage` as a persistent database.

## Features

- **User Authentication**: Secure registration and login system.
- **Product Management**: Admin interface to add, edit, and delete products.
- **Shopping Cart**: Fully functional cart with quantity adjustments and removal.
- **Responsive Design**: Optimized for mobile and desktop using Bootstrap 5.
- **Data Persistence**: All data (users, products, cart) is stored locally in the browser.

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Styling**: Bootstrap 5, Custom CSS
- **Storage**: Browser LocalStorage

## Getting Started

1. **Clone the repository** (or download the files).
2. **Open the application**:
   - Simply open `index.html` in your web browser.
   - For the best experience, use a local development server (e.g., VS Code Live Server).
